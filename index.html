Technical Analysis Script

Python script used for trend analysis this quarter.


import pandas as pd
import yfinance as yf

# Fetch quarterly stock data for ACME
acme_ticker = yf.Ticker("ACME")
q3_data = acme_ticker.history(period="3mo")

# Calculate 50-day moving average
q3_data['MA50'] = q3_data['Close'].rolling(window=50).mean()
print(q3_data.tail())
